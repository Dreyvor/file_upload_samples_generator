# Source Index

Use these sources as implementation and methodology references.

- OWASP File Upload Cheat Sheet: https://cheatsheetseries.owasp.org/cheatsheets/File_Upload_Cheat_Sheet.html
- PayloadsAllTheThings, Upload Insecure Files: https://github.com/swisskyrepo/PayloadsAllTheThings/blob/master/Upload%20Insecure%20Files/README.md
- Mitra by corkami: https://github.com/corkami/mitra
- Polydet polyglot database: https://github.com/Polydet/polyglot-database
- PortSwigger Web Security Academy, file upload vulnerabilities: https://portswigger.net/web-security/file-upload

Notes:

- Treat PayloadsAllTheThings as a methodology source, not as a payload source to blindly copy.
- Treat Mitra as an optional external dependency.
- Treat Polydet as optional reference data, not bundled content.
- Prefer generated benign samples over vendored payloads.
